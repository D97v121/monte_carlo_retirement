This model will simulate a given number of possible outcomes given a certain portfolio using a monte carlo model.
Success is defined as having a positive balance at the final year. 

INPUTS:
- Ranges
- Inputs for now
- Starting balance
- Expected return
- Volatility
- Withdrawal amount
- Inflation
- Number of years
        
OUTPUTS:
- Success probability
- Distribution of ending balances
- Worst-case years
- Median outcome
- Charts
